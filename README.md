# SISTEMA DE CADASTRO DE FUNCIONÁRIOS

## DESCRIÇÃO

Este projeto é desenvolvido em Python com o objetivo de praticar conceitos de programação, banco de dados e organização de código.

O sistema permite gerenciar funcionários por meio de um banco de dados SQLite, realizando operações de cadastro, consulta, atualização e remoção de registros.

## FUNCIONALIDADES

- Cadastro de funcionários
- Listagem de funcionários
- Busca por funcionário
- Atualização de dados
- Remoção de funcionários

## Tecnologias utilizadas

- Python 3
- SQLite
- sqlite3 (biblioteca padrão do Python)

## Estrutura do projeto

`projeto/
│
├── main.py
├── banco.py

└── README.md
`

## COMO EXECUTAR

1. Clone este repositório
2. Certifique-se de ter o Python 3 instalado.
3. Execute o arquivo `main.py`:

`python main.py`
